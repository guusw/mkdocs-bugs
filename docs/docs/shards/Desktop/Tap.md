# Desktop.Tap

<div class="sh-parameters" markdown="1">
| Name | - {: #sh-flags-row} | Description | Default | Type |
|------|---------------------|-------------|---------|------|
| `<input>` || | | `Int2` |
| `<output>` || | | `Int2` |
| `Window` | :fontawesome-solid-circle-plus:{title="Optional"}  | None or a window variable we wish to use as relative origin. | `None` | `Object ` |
